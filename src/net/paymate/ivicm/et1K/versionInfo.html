Information  Length  Type  Description 
Product ID  13  ASCII  String identifying the product. Will always be "5992". 
Application Version  14  ASCII  Version of the application currently running in the 5992.Operating 
System Version  11  ASCII  Version of the operating system currently in the 5992. 
Digitizer Version  11  ASCII  Version of the digitizer firmware. 
Flash Size  2  Binary (LO HI) Amount of flash memory, in kilobytes, currently installed. 
Video RAM Size  2  Binary (LO HI) Amount of video RAM, in kilobytes, currently installed. 
MSR Configuration 1  Binary Magnetic stripe reading hardware installed on the 5992.
Terminal Serial Number 21  ASCII  Unique serial number of 5992 
RAM Size  2  Binary (LO HI) Amount of volatile RAM, in kilobytes, currently installed. 
Link Library Version 8  ASCII  Version of the library linked with the application currently in the 5992. 
Security Module Version 8  ASCII  Version of the security module code currently running in the 5992's security processor. 
Downloaded Application Version 8  ASCII  Version of the file used to download the current application.
If the application is downloaded using the NCR style download, the version will be contained in the file used to download the application.
If the application is downloaded using CMDL, this version will be the same as the Application Version. 
Downloaded Parameter Version 8  ASCII  Version of the file used to download the current parameter set.
If the parameter set is downloaded using the NCR style download, this version will be contained in the parameter file. 
If the parameter set is downloaded using the CMDL, this version will be the version that is written as one of the parameters. 

Unless specifically noted above, each ASCII field above will be left justified and NUL (0x00) filled.

Color schemes (in standard RGB format & HTML format):

Color Scheme Dark color (header/footer) Medium color (alternating color) Light color (other alternating color and header font)
Heat (Red) 128,0,0 - 800000 192,128,128 - C08080 255,192,192 - FFC0C0
Money (Green) 0,128,0 - 008000 128,192,128 - 80C080 192,255,192 - C0FFC0
Tranquility (Blue) 0,0,128 - 000080 128,128,192 - 8080C0 192,192,255 - C0C0FF
Midnight (Grey) 0,0,0 - 000000 128,128,128 - 808080 192,192,192 - C0C0C0
Bronze 128,128,0 - 808000 192,192,128 - C0C080 255,255,192 - FFFFC0
Plum 128,0,128 - 800080 192,128,192 - C080C0 255,192,255 - FFC0FF
Teal 0,128,128 - 008080 128,192,192 - 80C0C0 192,255,255 - C0FFFF
Monochrome (just black and white) 0,0,0 - 000000 255,255,255 - FFFFFF 0,0,0 - 000000 (font must be white)
Simple (just gray and white) 255,255,255 - FFFFFF 192,192,192 - C0C0C0 255,255,255 - FFFFFF
Plain (no colors) 255,255,255 - FFFFFF 255,255,255 - FFFFFF 255,255,255 - FFFFFF

These color schemes can be mixed and matched, to some extent, allowing for more varied palettes, but sticking with only one or two at a time if we can is suggested, at least to start.  These are generated in code, so associates can define their own favorite color schemes, per MACID (per terminal / software key / dongle / whatever), or whatever.  This serves for easier browsing of transactions.

The color schemes are designed for readability by persons with certain vision impairments.
